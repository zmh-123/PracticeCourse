## 基础版算法实现及代码说明

### 1. 消息填充（padding）
- **数学原理**：
  - 输入消息 $m$，先补1，再补0，最后补64位原始长度（单位：比特），使总长度为512的倍数。
  - 公式：
     $m' = m \|\| 1 \|\| k*0 \|\| \text{len}(m)_{64}$
    其中 $k$使得 $\text{len}(m') \equiv 448 \pmod{512}$。
- **代码实现**：
  
  ```c++
  string padding(string str) {//对数据进行填充 
  	string res = "";
  	for (int i = 0; i < str.size(); i++) {//首先将输入值转换为16进制字符串
  		res += DecToHex((int)str[i]);
  	}
  	cout << "输入字符串的ASCII码表示为：" << endl;
  	for (int i = 0; i < res.size(); i++) {
  		cout << res[i];
  		if ((i + 1) % 8 == 0) {
  			cout << "  ";
  		}
  		if ((i + 1) % 64 == 0 || (i + 1) == res.size()) {
  			cout << endl;
  		}
  	}
  	cout << endl;
  	int res_length = res.size() * 4;//记录的长度为2进制下的长度
  	res += "8";//在获得的数据后面添1，在16进制下相当于是添加8
  	while (res.size() % 128 != 112) {
  		res += "0";//"0"数据填充
  	}
  	string res_len = DecToHex(res_length);//用于记录数据长度的字符串
  	while (res_len.size() != 16) {
  		res_len = "0" + res_len;
  	}
  	res += res_len;
  	return res;
  }
  ```
  
  

### 2. 消息分组
- **数学原理**：
  - 每512位（128十六进制字符）为一组，分组处理。
- **代码实现**：

  string iteration(string str) {//迭代压缩函数实现

### 3. 消息扩展（消息调度）
- **数学原理**：
  - $W[0..15] = M[0..15]$（原始分组）
  -  $[i] = P_1(W[i-16] \oplus W[i-9] \oplus (W[i-3] \lll 15)) \oplus (W[i-13] \lll 7) \oplus W[i-6], \quad i=16..67$ 
  -  $W'_i = W[i] \oplus W[i+4], \quad i=0..63$
  - 其中
     $P_1(X) = X \oplus (X \lll 15) \oplus (X \lll 23)$
- **代码实现**：
  
  ```c++
  string extension(string str) {//消息扩展函数
  	string res = str;//字符串类型存储前68位存储扩展字W值
  	for (int i = 16; i < 68; i++) {//根据公式生成第17位到第68位的W值
  		res += XOR(XOR(P1(XOR(XOR(res.substr((i-16)*8,8), res.substr((i - 9) * 8, 8)), LeftShift(res.substr((i - 3) * 8, 8), 15))), LeftShift(res.substr((i - 13) * 8, 8), 7)), res.substr((i - 6) * 8, 8));
  	}
  	cout << "扩展后的消息：" << endl;
  	cout << "W0,W1,……,W67的消息：" << endl;
  	for (int i = 0; i < 8; i++) {
  		for (int j = 0; j < 8; j++) {
  			cout << res.substr(i * 64 + j * 8, 8) << "  ";
  		}
  		cout << endl;
  	}
  	cout << res.substr(512, 8) << "  " << res.substr(520, 8) << "  " << res.substr(528, 8) << "  " << res.substr(536, 8) << endl;
  	cout << endl;
  	for (int i = 0; i < 64; i++) {//根据公式生成64位W'值
  		res += XOR(res.substr(i * 8, 8), res.substr((i + 4) * 8, 8));
  	}
  	cout << "W0',W1',……,W63'的消息：" << endl;
  	for (int i = 0; i < 8; i++) {
  		for (int j = 0; j < 8; j++) {
  			cout << res.substr(544+i * 64 + j * 8, 8) << "  ";

  		cout << endl;
  	}
  	cout << endl;
  	return res;
  }
  ```
  
  

### 4. 压缩函数（compress）
- **数学原理**：
  - 64轮迭代，A-H八个32位变量循环更新：
    
    -  $SS_1 = ((A \lll 12) + E + (T_j \lll j)) \lll 7$
    -  $SS_2 = SS_1 \oplus (A \lll 12)$
    -  $TT_1 = FF(A,B,C,j) + D + SS_2 + W'_j$
    -  $TT_2 = GG(E,F,G,j) + H + SS_1 + W_j$
    - 变量循环移位和赋值
  - 其中：
    -  $FF_j(X,Y,Z) = \begin{cases}X \oplus Y \oplus Z, & 0 \leq j \leq 15 \\(X \land Y) \lor (X \land Z) \lor (Y \land Z), & 16 \leq j \leq 63
    \end{cases}$
    -  $GG_j(X,Y,Z) = \begin{cases}
        X \oplus Y \oplus Z, & 0 \leq j \leq 15 \\
        (X \land Y) \lor (\lnot X \land Z), & 16 \leq j \leq 63
      \end{cases}$
    -  $P_0(X) = X \oplus (X \lll 9) \oplus (X \lll 17)$
    -  $T_j = \begin{cases}
        0x79CC4519, & 0 \leq j \leq 15 \\
        0x7A879D8A, & 16 \leq j \leq 63
      \end{cases}$
- **代码实现**：
  
  ```c++
  string compress(string str1, string str2) {//消息压缩函数
  	string IV = str2;
  	string A = IV.substr(0, 8), B = IV.substr(8, 8), C = IV.substr(16, 8), D = IV.substr(24, 8), E = IV.substr(32, 8), F = IV.substr(40, 8), G = IV.substr(48, 8), H = IV.substr(56, 8);
  	string SS1 = "", SS2 = "", TT1 = "", TT2 = "";
  	cout << "迭代压缩中间值: " << endl;
  	cout << "    A         B         C         D         E         F        G         H " << endl;
  	cout << A << "  " << B << "  " << C << "  " << D << "  " << E << "  " << F << "  " << G << "  " << H << endl;
  	for (int j = 0; j < 64; j++) {
  		SS1 = LeftShift(ModAdd(ModAdd(LeftShift(A, 12), E), LeftShift(T(j), (j%32))), 7);
  		SS2 = XOR(SS1, LeftShift(A, 12));
  		TT1 = ModAdd(ModAdd(ModAdd(FF(A, B, C, j), D), SS2), str1.substr((j + 68) * 8, 8));
  		TT2 = ModAdd(ModAdd(ModAdd(GG(E, F, G, j), H), SS1), str1.substr(j * 8, 8));
  		D = C;
  		C = LeftShift(B, 9);
  		B = A;
  		A = TT1;
  		H = G;
  		G = LeftShift(F, 19);
  		F = E;
  		E = P0(TT2);
  		cout << A << "  " << B << "  " << C << "  " << D << "  " << E << "  " << F << "  " << G << "  " << H << endl;
  	}
  	string res = (A + B + C + D + E + F + G + H);
  	cout << endl;
  	return res;
  }
  ```

## **优化策略**：

- 直接操作32位整数
- 使用位运算替代字符串转换
- 内联关键函数
- 循环展开
- 内存预取优化

## **关键优化点**

#### 1. 计算优化

- **位运算替代字符串转换**：

  ```c++
  // 替代字符串拼接
  uint32_t SS1 = ROTL((ROTL(A,12) + E + ROTL(T_j[j],j)), 7);
  ```

- **循环展开**：

  ```c++
  // 8步展开减少分支预测失败
  for (int i = 0; i < 64; i += 8) {
      // 第1步计算
      // ... 
      // 第8步计算
  }
  ```

#### 2. 内存优化

- **大端内存加载**：

  ```c++
  inline uint32_t load_be32(const uint8_t* b) {
      return (b[0]<<24) | (b[1]<<16) | (b[2]<<8) | b[3];
  }
  ```

- **预取机制**：

  ```c++
  __builtin_prefetch(next_block); // 预取下一数据块
  ```

#### 3. 指令级优化

- **内联关键函数**：

  ```c++
  __attribute__((always_inline)) 
  uint32_t FF(uint32_t x, uint32_t y, uint32_t z, int j) {
      return (j < 16) ? (x ^ y ^ z) : ((x & y) | (x & z) | (y & z));
  }
  ```
